# Классификация транспорта

## Описание проекта
Лабораторная работа по дообучению предобученных моделей из библиотеки `timm` для задачи классификации изображений транспортных средств.

**Задача**: Классификация изображений:
- 🚲 **bicycle** (велосипед)
- 🚗 **car** (машина)
- 🏍️ **motorcycle** (мотоцикл)

## Структура проекта
- `app/` — Gradio приложение для инференса модели
- `data/` — данные и их описание
- `models` - сохраненные модели
- `notebooks` - ноутбуки с экспериментами
- `src` - исходный код
- `requirements.txt` — зависимости проекта

## Установка
```bash
git clone https://github.com/vika890/lab1.git
cd lab1
pip install -r requirements.txt
```
## Для запуска приложения
``` bash
python3 app/app.py
```
