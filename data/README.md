## Набор данных

- **Общее количество**: 90 изображений (по 30 на класс)
- **Источник**: Собраны вручную из открытых источников (Freepik)
- **Разделение**: 
  - Train: 70% (63 изображения)
  - Validation: 15% (12 изображений)  
  - Test: 15% (15 изображений)

### Предобработка
- Resize: 224×224 пикселей
- Нормализация: ImageNet (mean=[0.485, 0.456, 0.406], std=[0.229, 0.224, 0.225])

### Аугментации (только train)
- Random Horizontal Flip (p=0.5)
- Random Rotation (±15°)
- Color Jitter (яркость, контрастность, насыщенность)
- Random Resized Crop (scale 0.8-1.0)

## Структура
- `data/raw/`: Исходные изображения, разделенные по классам.
- `data/split/`: Разделенные данные (train/val/test).
